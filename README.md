php_benchmark
===========


## What is it?

phpで作られた簡易ベンチマーク、のようです。[^1]  
大文字小文字のアルファベット＋0～9までの数字の合計62文字の中からランダムで10桁の文字を求めるプログラム。

## How to Use

```
$ git clone https://github.com/sogaoh/php_benchmark.git
$ cd php_benchmark
$ php php_benchmark.php

```
